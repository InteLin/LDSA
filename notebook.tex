mode means that a small scale cluster is simulated, running daemons on
the local machine and using HDFS.

    \subsubsection{Task 1.2}\label{task-1.2}

Setup hadoop as pseudo-distributed by going to
'/usr/local/hadoop/etc/hadoop/core-site.xml' and adding:

\begin{verbatim}
<configuration>
    <property>
        <name>fs.defaultFS</name>
        <value>hdfs://localhost:9000</value>
    </property>
</configuration>
\end{verbatim}

and to '/usr/local/hadoop/etc/hadoop/hdfs-site.xml':

\begin{verbatim}
<configuration>
    <property>
        <name>dfs.replication</name>
        <value>1</value>
    </property>
</configuration>
\end{verbatim}

These three commands were executed in order to get passphraseless
connection to localhost ssh:

\texttt{\$\ chmod\ 0600\ \textasciitilde{}/.ssh/authorized\_keys}

System was formatted using command
\texttt{\$\ bin/hdfs\ namenode\ -format} NameNode and DataNode daemons
were started by executing command \texttt{\$\ sbin/start-dfs.sh}

\begin{itemize}
\tightlist
\item
  Explain the roles of the different classes in the file WordCount.java.
